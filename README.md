# MCA Practical File

This repository is created for performing Git and GitHub lab exercises as part of MCA (TYP-2) curriculum.

## Objectives
- To understand version control using Git
- To learn GitHub repository management
- To perform collaboration using GitHub

## Experiments Covered
1. Repository Creation
2. Forking and Cloning
3. Making Changes (Commit & Push)
4. Creating Pull Request
5. Reviewing and Merging Pull Request
6. Git Installation and Configuration
7. Local Repository Creation
8. Branching and Merging
9. Pulling Updates

## Tools Used
- Git
- GitHub
